Comment utiliser MaizeDiseaseClassifier
MaizeDiseaseClassifier est une application Android qui identifie les maladies du maïs à partir d'images et fournit des recommandations agricoles en français. Suivez ces étapes pour utiliser l'application :

Installer l'application :
Téléchargez et installez l'application sur un appareil Android (version 7.0 ou supérieure).
Assurez-vous que l'appareil dispose d'une caméra fonctionnelle.
Lancer l'application :
Ouvrez l'application depuis l'écran d'accueil ou le menu des applications.
Accorder les permissions :
Lorsque l'application s'ouvre, elle peut demander des permissions :
Caméra : Autorisez l'accès pour prendre des photos.
Stockage (pour Android 13 ou inférieur) : Autorisez l'accès pour sélectionner des images depuis la galerie.
Cliquez sur Autoriser dans les invites de permission.
Capturer ou sélectionner une image :
Prendre une photo :
Cliquez sur le bouton Prendre une photo ("Take Photo").
Utilisez la caméra pour photographier une plante de maïs.
Confirmez la photo pour lancer l'analyse.
Choisir depuis la galerie :
Cliquez sur le bouton Choisir depuis la galerie ("Pick from Gallery").
Sélectionnez une image de maïs depuis votre galerie via le sélecteur de photos.
Confirmez pour lancer l'analyse.
Voir les résultats :
L'application analyse l'image et affiche :
Résultat : La maladie détectée ou l'état de la plante ("HEATHLY" pour sain, "MLN" pour nécrose létale du maïs, ou "MSV" pour virus de la striure du maïs).
Probabilités : Pourcentage de confiance pour chaque classe (par exemple, "HEATHLY : 85%").
Recommandations : Conseils en français, par exemple :
Sain : "La plante est saine. Continuez les bonnes pratiques agricoles : irrigation régulière, fertilisation équilibrée, et surveillance des nuisibles."
MLN : "Maize Lethal Necrosis (MLN) détecté. Recommandations : Retirez et détruisez les plantes infectées. Utilisez des semences résistantes et appliquez des insecticides pour contrôler les vecteurs (pucerons, cicadelles)."
MSV : "Maize Streak Virus (MSV) détecté. Recommandations : Éliminez les plantes infectées. Plantez des variétés résistantes et contrôlez les cicadelles avec des insecticides appropriés."
Répéter si nécessaire :
Prenez ou sélectionnez une autre image pour effectuer une nouvelle analyse.
Conseils
Qualité de l'image : Assurez-vous que l'image est claire, bien éclairée, et centrée sur la plante de maïs pour des résultats précis.
Connexion : Aucune connexion Internet n'est requise, car l'analyse est effectuée localement sur l'appareil.
Problèmes : Si l'application ne fonctionne pas (par exemple, crash ou erreur), vérifiez que les permissions sont accordées et que l'appareil est compatible (Android 7.0+). Contactez le support si nécessaire.
